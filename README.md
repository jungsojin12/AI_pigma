### 💡 사용 기술

* HTML, CSS, JavaScript
* 반응형 웹 디자인
* [사용된 주요 기술 명시]
1. 스크롤시 상단에 픽스되는 FixHeader영역제작
2. 애니메이션이 적용된 이미지슬라이드 제작 - Slick플러그인 적용법
3. 스크롤의 위치에 반응하는 애니메이션 제작-Scrolla플러그인 적용법
4. 햄버거 메뉴 버튼 클릭 시 슬라이드 되는 gnbMenu
5.Top 버튼을 클릭시 상단으로 부드럽게 이동하는 Script
6.Media Query를 활용한 반응형 웹제작
